# cyber-security-basics

##  Description:

This is a simple Python password checker script that checks whether a password is strong or weak.

---

##  How to Run:

Make sure Python is installed on your system.

Run the following command in terminal:

```bash
python password-checker.py
```


##Requirements:

Python 3.x

## Example:

Input:
Enter password: MyPass123

Output:
Strong Password

Output:

Strong Password
