# cyber-security-basics

## Introduction to Cyber Security

### What is Cyber Security?

Cyber security is the practice of protecting computers, servers, mobile devices, networks, and data from digital attacks. Think of it like locking the doors and windows of your house — but for your digital life.

### Why is Cyber Security Important?

Every day, billions of devices are connected to the internet, and with that connectivity comes risk:

- **Protecting personal data**: Your passwords, bank details, and private messages need protection
- **Preventing financial loss**: Cyber attacks cost businesses and individuals billions each year
- **Maintaining trust**: Companies that suffer data breaches lose customer confidence
- **National security**: Critical infrastructure like power grids and hospitals depend on secure networks

> **Example:** In 2017, the WannaCry ransomware attack affected over 200,000 computers across 150 countries, crippling hospitals, banks, and businesses.

### Main Types of Cyber Threats

| Threat Type | Description | Real-World Example |
|---|---|---|
| **Malware** | Malicious software that damages or takes control of your system | Viruses, trojans, spyware |
| **Phishing** | Fake emails or messages that trick you into revealing sensitive info | "Your account has been compromised — click here to reset your password" |
| **Ransomware** | Software that locks your files and demands payment to unlock them | WannaCry, CryptoLocker |
| **Social Engineering** | Manipulating people into giving up confidential information | A caller pretending to be IT support asking for your login |
| **DDoS Attacks** | Overwhelming a website with traffic to make it crash | Attackers flooding an online store during a sale event |

### Basic Security Concepts

#### 1. The CIA Triad

The foundation of cyber security rests on three principles:

- **Confidentiality**: Only authorized people can access data (e.g., passwords, encryption)
- **Integrity**: Data is accurate and hasn't been tampered with (e.g., checksums, digital signatures)
- **Availability**: Systems and data are accessible when needed (e.g., backups, redundancy)

#### 2. Defense in Depth

No single security measure is enough. Use multiple layers of protection:

```
Your Data
├── Encryption
├── Access Controls (passwords, 2FA)
├── Firewall
├── Antivirus
└── Physical Security (locked doors)
```

#### 3. Authentication vs. Authorization

- **Authentication** answers "Who are you?" — Username + password, fingerprint, security key
- **Authorization** answers "What can you do?" — Read-only access, admin privileges

#### 4. Common Best Practices

- Use strong, unique passwords (consider a password manager)
- Enable two-factor authentication (2FA) wherever possible
- Keep software and operating systems updated
- Be suspicious of unexpected emails and links
- Back up important data regularly
- Don't use public Wi-Fi without a VPN for sensitive activities
